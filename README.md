 # Regressão Linear em Python

Projeto simples de Machine Learning utilizando regressão linear com a biblioteca scikit-learn.

## Tecnologias
- Python
- scikit-learn

## Objetivo
Treinar um modelo capaz de prever valores com base em dados de entrada.

## Exemplo
O modelo aprende uma relação linear (y = 2x) e realiza previsões com base nisso.
